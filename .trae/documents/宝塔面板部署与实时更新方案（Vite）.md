## 现在先准备什么（你搞域名/宝塔时顺手做）
- 备案/解析：域名 A 记录指向服务器公网 IP。
- 宝塔基础：装好 Nginx（或 OpenLiteSpeed）+ Git + Node.js。
- 开放端口：80/443 放行；准备好 HTTPS 证书（宝塔可一键申请）。

## 等你回来部署时做什么（按顺序照做）
1. 服务器上把代码 clone 下来。
2. npm install（或 npm ci）。
3. 配好 .env（如有）。
4. npm run build 生成 dist。
5. 宝塔“网站”根目录指向 dist。
6. 加 SPA 重写规则，避免刷新 404。

## 想要“提交即更新”（你回来后选一个）
- Webhook 自动部署：提交代码 → 服务器脚本自动 git pull + build + 覆盖 dist（推荐）。
- 计划任务定时部署：每隔几分钟自动拉取并构建（简单但不算实时）。
